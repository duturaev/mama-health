# Здоровье мамы — PWA

Инструкция по установке на телефон через GitHub Pages.

## Шаг 1 — Создайте GitHub аккаунт
Перейдите на https://github.com и зарегистрируйтесь (если нет аккаунта).

## Шаг 2 — Создайте новый репозиторий
1. Нажмите "+" → "New repository"
2. Название: `mama-health`
3. Выберите "Public"
4. Нажмите "Create repository"

## Шаг 3 — Загрузите файлы
Загрузите все файлы из этой папки:
- index.html
- manifest.json
- sw.js
- icons/icon-192.png
- icons/icon-512.png

## Шаг 4 — Включите GitHub Pages
1. Откройте репозиторий → Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: main / root
4. Нажмите Save

Через 1–2 минуты приложение будет доступно по адресу:
https://ВАШ_ЛОГИН.github.io/mama-health/

## Шаг 5 — Установите на телефон

### Android (Chrome):
1. Откройте ссылку в Chrome
2. Появится баннер "Добавить на главный экран" — нажмите
3. Или: меню (⋮) → "Добавить на главный экран"

### iPhone (Safari):
1. Откройте ссылку в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой)
3. Выберите "На экран «Домой»"

## Обновление данных
Когда Claude выдаст новый index.html:
1. Откройте репозиторий на GitHub
2. Нажмите на index.html → карандаш (редактировать)
3. Вставьте новый код → Commit changes
4. Через минуту приложение обновится автоматически
